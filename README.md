\# Selenium TestNG Automation Project



This project contains basic Selenium WebDriver automation examples using TestNG.



\## 🔧 Technologies Used

\- Java

\- Selenium WebDriver

\- TestNG

\- Maven



\## 📂 Project Structure

\- `src/main/java/FirstClass/` → All test classes

\- `pom.xml` → Dependencies

\- `testng.xml` → Test execution configuration



\## 🚀 Features

\- Browser automation

\- Locator strategies (ID, Name, Class, XPath)

\- TestNG annotations

\- Basic interaction handling



\## 📌 Notes

\- Driver executables are not included

\- Uses Chrome browser for testing
